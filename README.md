# Raspberry-PI2040-Custom


## Custom RP2040 Development Board

This project is a **custom-designed development board based on the Raspberry Pi RP2040 microcontroller**, built for embedded systems experimentation, firmware development, and learning PCB design. The board integrates the RP2040 dual-core ARM Cortex-M0+ microcontroller with carefully designed power management, USB connectivity, and accessible GPIO pins.

A key feature of the board is its **dual power capability**, allowing it to operate either from **USB power or a rechargeable battery source**. The power system includes a regulated **3.3 V supply**, proper decoupling, and power switching circuitry to ensure stable operation in both modes.

The PCB layout focuses on **good power integrity and signal routing practices**, including close placement of decoupling capacitors, controlled USB differential traces, and optimized trace widths for power delivery.

### Features

* **RP2040 dual-core ARM Cortex-M0+ microcontroller**
* **USB interface** for programming and communication
* **Battery-powered operation** for portable use
* **3.3 V regulated power supply**
* **Automatic or selectable power input (USB / battery)**
* **Exposed GPIO pins** for sensors and peripherals
* **Proper decoupling and power filtering**
* **Compact custom PCB layout**

### Purpose

This project is designed as a **learning and prototyping platform** for:

* Embedded firmware development
* Custom microcontroller board design
* Power management in portable electronics
* PCB layout and hardware debugging
